# Python Basics

## Description
This repository contains beginner-friendly Python programs created as part of my learning journey in Computer Science (CSE, AIML). These projects demonstrate basic programming skills, problem-solving, and understanding of Python fundamentals.

## Projects Included

1. **Calculator**
   - A simple command-line calculator that performs addition, subtraction, multiplication, and division.
   
2. **To-Do List**
   - A command-line application to add, view, and delete tasks. Demonstrates lists, loops, and functions.
   
3. **Dice Rolling Simulator**
   - A fun program that simulates rolling a dice using Python's random module.
   
4. **Mad Libs Game**
   - A small interactive program that generates a funny story based on user input. Demonstrates string manipulation and f-strings.

## Skills Demonstrated
- Python basics: variables, loops, conditionals, functions
- Input/output and user interaction
- Lists and string manipulation
- Random number generation

## How to Run
1. Clone the repository:
'''git clone https://github.com/suhela-solkar/python-basics.git'''
2. Navigate into the folder:
3. cd python-basics
4. Run any Python file:
   python calculator.py
   python to_do_list.py
   python dice_simulator.py
   python mad_libs.py
